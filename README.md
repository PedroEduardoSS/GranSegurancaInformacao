# Gran Segurança da Informação
Repositório de estudos e interação para o Curso de Segurança da Informação da Gran Faculdade

## Boas-vindas!
Esse repositório está destinado para organizar projetos voltados para segurança da informação para graduandos da gran faculdade, mas aberto para todos que gostam ou são entusiastas da área.
Também é um espaço de interação e compatilhamento de conhecimento entre os participantes por meio da aba Issues ou Discussions que serão abordadas no próximo tópico.

## Organização do repositório
1. Code: Nessa aba estão pastas e arquivos de códigos que representam projetos ou laboratórios para praticar conteúdos técnicos.
2. Issues: Voltado para reportar bugs ou sugestões para melhoria para o repositório ou projeto.
3. Discussions: Nessa aba é focado para interação dos participantes, seja dúvidas em disciplinas, anúncios, amostras para mostrar sua evolução, Perguntas e Respostas ou ideias
